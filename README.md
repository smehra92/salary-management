# Salary Management

Web-based salary management software for an HR manager overseeing ~10,000
employees across multiple countries. Replaces the current Excel-based workflow
and answers questions about how the organization pays its people.

## Tech Stack
- **Backend:** Node.js, TypeScript, Express
- **Database:** SQLite (via Prisma)
- **Frontend:** React, TypeScript (Vite), shadcn/ui
- **Testing:** Vitest

## Structure
- `server/` — REST API, domain logic, tests
- `client/` — React UI
- `docs/`   — requirements, architecture, trade-offs
- `ai/`     — prompts and AI-tooling notes
